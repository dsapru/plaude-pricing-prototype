# Plaude Pricing Prototype

A modern, responsive pricing page prototype for Plaude. It features a sleek design, micro-animations, and interactive elements built with HTML and vanilla CSS.

## Features
- **Modern Aesthetics:** Glassmorphism, dynamic gradients, and smooth hover effects.
- **Responsive Layout:** Optimized for desktop, tablet, and mobile devices.
- **Interactive Toggles:** Easy switching between monthly and annual billing options.

## Getting Started
Simply open `index.html` in any modern web browser to view the prototype. No build step required.

## Deployment
This project consists of static files and is designed to be easily deployed using **GitHub Pages**, Vercel, or Netlify.
